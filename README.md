# 📒 Premium Contact Book Manager

A modern Contact Book application built using HTML, CSS, and JavaScript.

## 🚀 Features

- ➕ Add New Contacts
- ✏️ Edit Existing Contacts
- 🗑️ Delete Contacts
- 🔍 Search Contacts Instantly
- 📱 Phone Number Validation
- 📧 Email Validation
- 💾 Local Storage Support
- 📊 Contact Counter
- 🎨 Modern Glassmorphism UI
- 📱 Fully Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 📸 Project Preview

Premium Contact Book allows users to save and manage personal contacts efficiently with a clean and modern user interface.
